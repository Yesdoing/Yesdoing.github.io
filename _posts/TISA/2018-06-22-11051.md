## 문제

자연수 N과 정수 K가 주어졌을 때 이항 계수 (N K)를 10,007로 나눈 나머지를 구하는 프로그램을 작성하시오.


## 입력

첫째 줄에 N과 K가 주어진다. (1 ≤ N ≤ 1,000, 0 ≤ K ≤ N)


## 출력

 (N K)를 10,007로 나눈 나머지를 출력한다.



### 예제입력
```
5 2
```

### 예제 출력
```
10
```
